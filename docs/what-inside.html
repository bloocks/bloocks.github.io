What's inside Bloocks.

Bloocks is a set of different technologies integrated together to speed up the creation of high quality websites.

By getting an effective and free starting point to start testing your business ideas in the battlefield.

With what's tools Bloocks is made

Bloocks is a hybrid generator of static websites, it has been assembled using a series of open source tools, which, assembled together, create a complete development structure to create websites.

Frontend tools

Open source, frontend frameworks and libraries, that Bloocks use to render the generated web pages.

Dev tools

Open source, developer backend tools, that bloocks use to build your website and make it production ready.

Hosting
